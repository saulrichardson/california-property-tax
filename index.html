This map was created by Saul Richardson and Victoria Michalska. We aim to provide a clearer understanding of Los Angeles' property tax landscape, highlighting how property values and tax bills interact across the city. Our goal is to enhance public awareness about property tax dynamics and to facilitate informed discussions on property tax policies.

The code behind this map is available on GitHub. The data used to generate the map comes from official Los Angeles sources. For more details on the data, including its most recent updates, please visit our GitHub repository.
